# Bank Loan Request Management System

This project is a Java Swing desktop application developed as a learning prototype during my first internship in a banking environment.

## Features

- Register client information
- Select loan type
- Calculate estimated monthly payment
- Add loan requests to a table
- Search requests by CIN
- Approve or reject requests
- Delete requests
- Save data into a CSV file

## Technologies Used

- Java
- Java Swing
- JTable
- JComboBox
- File handling with CSV

## Purpose

The purpose of this project is to practice Java Swing, event handling, input validation, table management, and file handling while creating a simple software solution inspired by banking workflows.

This project is for educational purposes only.
